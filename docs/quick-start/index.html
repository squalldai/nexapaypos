Quick Start Guide

Get up and running with NexaPay API in 5 minutes

1

Create Your Account

Sign up for a NexaPay account at nexapay.com. You'll receive access to the merchant dashboard where you can manage your API credentials.

  • Free 14-day trial with full API access
  • No credit card required for trial
  • Access to sandbox environment
2

Generate API Key

Navigate to Settings → API Keys in your dashboard. Click "Create New Key" to generate your API credentials.

API KeyLive
nxpos_live_sk_xxxxxxxxxxxxxxxxxxxxxxxx

Important: Store your API key securely. Never share it or commit it to version control.

3

Install SDK (Optional)

Use our official SDKs for faster integration, or make direct API calls with any HTTP client.

Node.js
npm install @nexapay/sdk
Python
pip install nexapay
4

Make Your First Request

Test your integration by fetching your store information:

cURL
curl -X GET "https://api.nexapay.com/v1/stores" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
Node.js
import NexaPay from '@nexapay/sdk';

const client = new NexaPay('YOUR_API_KEY');

const stores = await client.stores.list();
console.log(stores);
Python
from nexapay import NexaPay

client = NexaPay('YOUR_API_KEY')

stores = client.stores.list()
print(stores)
