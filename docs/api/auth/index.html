Detailed Examples

POST/auth/login

Authenticate user with email and password. Returns access token and refresh token.

Request Body

{
  "email": "user@example.com",
  "password": "secure_password",
  "remember_me": true,
  "device_info": {
    "user_agent": "Mozilla/5.0...",
    "ip_address": "192.168.1.1"
  }
}

Response

{
  "success": true,
  "data": {
    "user": {
      "id": "usr_abc123",
      "email": "user@example.com",
      "name": "John Doe",
      "role": "admin",
      "merchant_id": "mer_xyz789"
    },
    "tokens": {
      "access_token": "eyJhbGciOiJIUzI1NiIs...",
      "refresh_token": "ref_abc123xyz...",
      "expires_in": 86400
    },
    "session": {
      "id": "sess_123",
      "created_at": "2024-01-15T10:30:00Z",
      "expires_at": "2024-01-16T10:30:00Z"
    }
  }
}
POST/auth/api-keys

Create a new API key with specified scopes and expiration.

Request Body

{
  "name": "Production Integration",
  "type": "secret",
  "scopes": [
    "read_products",
    "write_orders",
    "read_customers"
  ],
  "expires_in_days": 90,
  "allowed_ips": ["192.168.1.0/24"],
  "metadata": {
    "environment": "production",
    "project": "web-store"
  }
}

Response

{
  "success": true,
  "data": {
    "id": "key_abc123",
    "name": "Production Integration",
    "key": "nxpos_live_sk_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
    "type": "secret",
    "scopes": ["read_products", "write_orders", "read_customers"],
    "created_at": "2024-01-15T10:30:00Z",
    "expires_at": "2024-04-15T10:30:00Z",
    "last_used_at": null
  }
}
POST/auth/refresh

Exchange refresh token for new access token.

Request Body

{
  "refresh_token": "ref_abc123xyz..."
}

Response

{
  "success": true,
  "data": {
    "access_token": "eyJhbGciOiJIUzI1NiIs...",
    "token_type": "Bearer",
    "expires_in": 86400
  }
}
